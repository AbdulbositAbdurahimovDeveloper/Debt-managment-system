Transaction API Documentation

Barcha tranzaksiyalar USD (Oltin ko'prik) mantiqi asosida ishlaydi.

usdAmount = amount / marketRate balanceEffect = usdAmount * clientRate
SALE (Sotuv) Mijoz qarzi oshadi (-)

Mijoz 12,800,000 so'mlik tovar oldi. Balansi AED da.

{
  "clientId": 1,
  "type": "SALE",
  "transactionCurrency": "UZS",
  "amount": 12800000,
  "marketRate": 12800,
  "clientRate": 3.67,
  "items": [
    { "productId": 10, "quantity": 1, "unitPrice": 1000 }
  ]
}
PAYMENT (To'lov) Mijoz qarzi kamayadi (+)

Mijoz AED da qarzini to'ladi.

{
  "clientId": 1,
  "type": "PAYMENT",
  "transactionCurrency": "AED",
  "amount": 3670,
  "marketRate": 3.67,
  "clientRate": 3.67,
  "description": "Kassa orqali AED olindi"
}
PURCHASE (Xarid) Bizning qarzimiz oshadi (+)

Ta'minotchidan 5000$ lik tovar keldik.

{
  "clientId": 5,
  "type": "PURCHASE",
  "transactionCurrency": "USD",
  "amount": 5000,
  "marketRate": 1,
  "clientRate": 1,
  "items": [
    { "productId": 12, "quantity": 5, "unitPrice": 1000 }
  ]
}
PURCHASE_PAYMENT Bizning qarzimiz kamayadi (-)

Ta'minotchiga 25.6 mln so'm to'ladik.

{
  "clientId": 5,
  "type": "PURCHASE_PAYMENT",
  "transactionCurrency": "UZS",
  "amount": 25600000,
  "marketRate": 12800,
  "clientRate": 1
}
TRANSFER (P2P O'tkazma) Sender: (-) | Receiver: (+)

Kuryer (ID:2) olingan pulni Ta'minotchiga (ID:5) topshirdi.

{
  "clientId": 2,
  "receiverClientId": 5,
  "type": "TRANSFER",
  "transactionCurrency": "UZS",
  "amount": 12800000,
  "marketRate": 12800,
  "clientRate": 1,
  "receiverRate": 1,
  "description": "Kuryerdan Ta'minotchiga o'tkazildi"
}
RETURN (Vozvrat) Mijoz qarzi kamayadi (+)

Mijoz 1000$ lik tovarni qaytardi.

{
  "clientId": 1,
  "type": "RETURN",
  "transactionCurrency": "USD",
  "amount": 1000,
  "marketRate": 1,
  "clientRate": 3.67,
  "items": [{ "productId": 10, "quantity": 1, "unitPrice": 1000 }]
}
RETURN_PAYMENT Mijoz qarzi oshadi (-)

Mijozga pulini qaytardik.

{
  "clientId": 1,
  "type": "RETURN_PAYMENT",
  "transactionCurrency": "USD",
  "amount": 500,
  "marketRate": 1,
  "clientRate": 3.67
}
